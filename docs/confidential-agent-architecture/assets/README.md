# Diagram Assets

This directory contains the rendered diagram assets used by the Confidential Agent architecture whitepaper.

- `*.svg` files are the editable source diagrams.
- `*.png` files are exported renderings referenced by the Markdown article.
- The diagrams intentionally use explicit architecture labels instead of generic decorative imagery, so protocol direction, attestation semantics, and project boundaries remain reviewable.

When updating a diagram, edit the matching SVG first and regenerate the PNG from that source.
